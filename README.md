# TheLinuxWays

对于有一定Linux操作基础的用户, 快速速通Linux嵌入式平台的生态的cheatsheet式资料.  

> 你还在看着手里的Linux开发板发呆?  
> 你还在觉得Linux内核是个遥不可及的大工程?  
> 你还在觉得外面的资料过于基础, 而你又不知道如何深入?  
> 
> 那么让我来快速帮你补课! 和那些课本上教给你的点灯教程**不一样**!

因为内容较为分散而且杂, 想到啥写啥, 没有固定的章节顺序. 但是我会尽力保证质量.  

**本文面向进阶用户, 因此, 默认你知道如何使用基础的Linux命令, 如ls, 如cd, 知道什么是gcc编译器. 同时, 你至少要能看懂c语言. 否则, 你需要自行补充一些前置知识.**
